Guia de Estudos Ativos CPA-10 — NotebookLM
Este projeto utiliza o NotebookLM como uma ferramenta de Inteligência Artificial Generativa aplicada à preparação para a certificação CPA-10 (ANBIMA). O objetivo é organizar, sintetizar e explicar o conteúdo regulatório e técnico do Sistema Financeiro Nacional. Todo o conteúdo é processado a partir de fontes oficiais, garantindo precisão técnica para o exame e para a prática profissional.

O NotebookLM desenvolvido neste projeto pode ser acessado aqui:
(https://notebooklm.google.com/notebook/f64ef93f-0eac-46a5-9358-3d75a4846963)

Objetivo do Projeto
Criar um ecossistema de aprendizagem ativa capaz de:

Sintetizar os 7 módulos da certificação ANBIMA de forma hierárquica.

Esclarecer normas de conduta, ética e prevenção à lavagem de dinheiro.

Diferenciar produtos de investimento (Renda Fixa, Variável e Previdência).

Auxiliar na interpretação de diretrizes da CVM e do Banco Central.

Funcionalidades do Notebook
Resumos Estruturados: Síntese técnica por módulo.

Glossário de Precisão: Definições de termos como SELIC, IPCA, FGC e Suitability.

Matrizes Comparativas: Tabelas entre PGBL/VGBL e regimes de tributação.

Simulados Estratégicos: Questionários baseados no conteúdo das apostilas.

Guia de Prompts: Comandos otimizados para revisão de conceitos complexos.

Fontes
As respostas são geradas exclusivamente a partir das fontes carregadas no repositório /fontes, que incluem:

Apostilas oficiais preparatórias para CPA-10.

Manuais de Ética e Autorregulação da ANBIMA.

Normativas atualizadas sobre o Sistema Financeiro Nacional.

Tecnologias Utilizadas
NotebookLM: Gestão de conhecimento
